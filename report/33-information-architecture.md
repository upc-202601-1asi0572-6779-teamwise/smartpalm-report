# 5.2. Information Architecture

La arquitectura de información de SmartPalm fue estructurada para facilitar el acceso rápido a datos agronómicos, alertas y herramientas de monitoreo dentro de un entorno de navegación intuitivo. La organización de contenidos se desarrolló considerando los dos segmentos principales del sistema: productores agrícolas e ingenieros agrónomos.

La estructura priorizó la reducción de complejidad operativa y la rápida localización de información crítica relacionada con sensores, condiciones climáticas, plantaciones y reportes técnicos.

Asimismo, se diseñó una arquitectura escalable que permite incorporar nuevas funcionalidades sin alterar significativamente la experiencia de navegación existente.
